**Project Overview**
This repository contains a practice data analytics project developed using Power BI to analyze road accident data and understand accident patterns, casualty distribution, and risk factors across different conditions. The dashboard transforms raw accident data into interactive visual insights to help explore accident severity, vehicle involvement, road conditions, and geographic accident distribution.

This project was created as part of data analytics and visualization practice, focusing on building an effective Power BI dashboard for public safety insights and data storytelling.

**Objective**

The objective of this project is to practice:

Building interactive dashboards using Power BI

Analyzing road accident and casualty data

Understanding factors influencing accidents such as road type, light conditions, and location

Identifying high-risk accident conditions

Presenting insights through clear visual storytelling

**Tools & Technologies Used**

Power BI – Data visualization and dashboard development

Microsoft Excel – Dataset source

DAX (Data Analysis Expressions) – Calculated measures and KPIs

Data Modeling – Structuring accident datasets for analysis

**Dataset**

The dataset used in this project was downloaded from kaggle that contains road accident and casualty information, including multiple attributes related to accident severity and environment.

Key fields in the dataset include:

Accident Date

Accident Severity

Number of Casualties

Number of Vehicles

Vehicle Type

Road Type

Urban / Rural Area

Light Conditions

Weather Conditions

Road Surface Conditions

Location Coordinates

Police Force / Region

The data was used only for learning, analysis, and visualization practice purposes.

**Key Insights**

From the dashboard analysis, several observations can be made:

Cars account for the majority of accident casualties.

Single carriageway roads show the highest accident involvement.

Urban areas experience significantly more casualties than rural areas.

Most accidents occur during daylight hours.

Accident distribution across the map reveals clusters in densely populated regions.
